# 🌬️ Fan Project – Pomodoro Breeze  
**STM32 기반 집중력 관리 선풍기**

## 📌 프로젝트 소개

**Pomodoro Breeze**는  
짧고 명확한 집중 구간을 위해 설계된 **타이머형 집중력 관리 선풍기**입니다.

요즘 사람들은 긴 시간 집중하기 어렵습니다.  
그래서 우리 팀은 **필요한 순간에만 바람을 제공하고,  
집중이 흐트러지기 전에 자동으로 멈추는 선풍기**를 만들었습니다.

---

## 🔄 동작 흐름 요약

1. 버튼 입력 감지
2. 동작 모드 설정 (50% / 100% / 회전)
3. LED 상태 표시
4. Timer 시작
5. 10초 후 자동 Stop + LED 종료 표시

---

## 🔘 버튼 구성 및 동작

| 버튼 | 기능 | 설명 |
|----|----|----|
| 1 | 정지 | 집중 종료 |
| 2 | Fan 50% | 가벼운 집중 (독서, 필기) |
| 3 | Fan 100% | 강한 집중 (코딩, 작업) |
| 4 | 회전 | 머리가 막힐 때 환기 모드 |

- 2번 / 3번 버튼 → 풍량에 따라 동작
- 4번 버튼 → 서보모터 기반 회전 동작
- **모든 동작은 LED로 상태 표시**

---

## ⏱️ 자동 Stop 기능

- 모든 동작은 **10초 후 자동 정지**
- 사용자가 직접 멈추지 않아도:
  - 소음 감소
  - 전력 절약
  - 집중 방해 최소화

---

## 💡 LED 상태 표시

- 현재 동작 상태 시각화
- 멀리서도 확인 가능
  - 지금 집중 중인지
  - 이미 종료되었는지

---

## ⚙️ 시스템 구성

- **MCU**: STM32
- **개발 환경**: STM32CubeIDE
- **입출력 구성**
  - Button 입력
  - LED 출력
  - PWM 제어
  - Timer 기반 동작
  - UART (DMA 방식)

---

## 🧩 주요 모듈 구조

### 📁 Core/Src
- `main.c` – 메인 상태 제어
- `button.c` – 버튼 디바운싱 및 이벤트 처리
- `led.c` – LED 상태 표현
- `servo.c` – 서보모터 제어
- `fnd.c` – Timer 기반 FND 제어
- `usart.c` – DMA 기반 UART 통신
